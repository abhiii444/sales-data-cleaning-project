# 📊 Sales Data Cleaning & Preprocessing Project

## 🧾 Description
This project focuses on cleaning and preprocessing a messy sales dataset using **Python (Pandas & NumPy)**. It transforms raw data with missing values, duplicates, and inconsistencies into a **clean, analysis-ready dataset**.

---

## 🚀 Highlights
✔️ Generated messy dataset  
✔️ Cleaned & standardized data  
✔️ Handled missing values & duplicates  
✔️ Fixed data types & date formats  
✔️ Applied outlier detection (IQR)

## 📌 Dataset Issues
- Missing values  
- Duplicate records  
- Inconsistent formats  
- Invalid values (age, rating, amount)  

---

## 🔧 Cleaning Steps
1. Data inspection (shape, types, nulls)  
2. Column renaming (lowercase, underscores)  
3. Duplicate removal  
4. Standardization (gender, country, category, payment)  
5. Date formatting (dd-mm-yyyy)  
6. Data type correction  
7. Missing value handling (median/mode)  
8. Outlier treatment (IQR method)
